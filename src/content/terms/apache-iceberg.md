---
title: "Apache Iceberg"
description: "A high-performance open table format for huge analytic datasets."
related: ["data-lakehouse"]
---

**Apache Iceberg** is an open table format originally developed at Netflix. It brings SQL tables to big data, enabling multiple engines to safely work with the same tables simultaneously with ACID transactions.

Iceberg is the foundational layer for modern Data Lakehouses, providing features like schema evolution, hidden partitioning, and time travel. Dremio has first-class support for Iceberg, optimizing query performance and simplifying data management.
